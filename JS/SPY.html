Welcome to Part 9 : Your JavaScript Level One Project !

For this project you will be building a generic website that will seem to ask mundane questions to normal users, but you secretly are looking for a spy! Anyone visiting your website will be asked a series of questions, only the Spy will be able to give specific answers you expect . If all questions are answered correctly, you will post a secret message in console for the spy to read !

Here are the four conditions you will check for the spy:
  • The Spy has the same first letter of his/her First Name and the Last Name.
  • The Spy is between the Age of 20 and 30(exclusive 20 & 30).
  • The Spy is atleat 170 cm tall.
  • The Spy has a nick name that ends with the letter 'y'.

We are going to leave this project a little more open-ended,but if you feel like you need more guidance,check out the solution walkthrough videos! But Believe it or not, we've actuallycovered everything you need to know to complete this task. I would suggest you use Javascript to prompt the user for information and set these to variables. THen use these variables to check for the four conditions to all be true. It's up to you what the secret message is !!!

A Spy with the FOllowing information should see your secret message:
  • Devil Digvijay
  • 20 years old
  • 178cm tall
  • Nick name is "Joy"

Keep in mind you ask the question prompts in whatever order you want, you can even seperate asking for first name and then another prompt for last name.
